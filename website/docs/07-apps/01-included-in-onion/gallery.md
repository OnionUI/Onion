# Gallery


A viewer for your screenshots.

Shortcuts:

| Button           | Function                                 |
| ---------------- | ---------------------------------------- |
| D-pad right or A | Next image                               |
| D-pad left or B  | Previous image                           |
| Y                | Toggle visibility of top and bottom bars |
| Menu             | Exit                                     |
